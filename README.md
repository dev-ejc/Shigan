# Shigan
An equity market portfolio application with performance and research features!

## Purpose:
The following is a full stack personal stock portfolio application to manage, buy and sell stocks.

## Stack:
Backend:
  Database: MongoDB
  ORM: Mongoose
  Framework: Express
Frontend:
  State Management: Redux
  Framework: React
